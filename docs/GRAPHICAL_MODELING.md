# Athena Graphical Model

This folder contains the current technical diagrams for Athena:

- `athena_architecture.mmd` — high-level system architecture
- `athena_runtime_sequence.mmd` — runtime decision flow and diagnostics path

Suggested workflow:

1. Open the `.mmd` files in VS Code.
2. Preview them with the Mermaid extension.
3. Update them whenever signal, risk, or observability flow changes.
